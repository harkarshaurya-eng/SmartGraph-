# SmartGraph
SmartGraph is a Python desktop app for visual double and triple integration with graph-based explanations and step-by-step symbolic output.
Concepts used: definite integration, double integration, triple integration, area, volume, accumulated value, Tkinter, and SymPy.


How to start the program:
1)Make a folder on your desktop named "SmartGraph".
2)press windows button on your compute(🪟)
3)type "powershell"
4)press enter
5)type this exact command 
```
git clone https://github.com/harkarshaurya-eng/SmartGraph-.git
cd SmartGraph-
pip install -r requirements.txt
python main.py
```
